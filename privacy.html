Politique de confidentialité

Cette page décrit le fonctionnement de la version bêta d’Oenaris. Dernière mise à jour : 19 juillet 2026.

Données de compte

La création et la connexion au compte sont assurées par Supabase Auth. Oenaris traite notamment votre adresse email, votre identifiant de compte et, si vous le renseignez, votre nom affiché. Le mot de passe est traité par Supabase et n’est pas stocké dans le code de l’application.

Données de cave

Les bouteilles, mouvements, emplacements, listes d’achat, notes de dégustation et préférences sont associés à votre espace Oenaris sur cet appareil. Lorsque la synchronisation est disponible et utilisée, une copie peut être envoyée vers le projet Supabase d’Oenaris.

Exports et données techniques

Vous pouvez exporter votre cave en JSON, CSV ou PDF. L’application peut aussi conserver sur cet appareil un journal d’erreurs et des informations de diagnostic, par exemple la version, le navigateur et la taille de stockage estimée. Ces éléments ne sont transmis que lorsque vous déclenchez explicitement un export ou une action de synchronisation prévue.

Services externes

  • Supabase fournit l’authentification et les fonctions de sauvegarde configurées.
  • GitHub Pages héberge les fichiers du site public.
  • Une fonction d’intelligence artificielle peut être utilisée lorsqu’elle est configurée côté serveur ; aucune clé API secrète n’est intégrée au navigateur.

Paiement

Aucun paiement n’est actif pendant cette bêta et Oenaris ne demande aucune donnée bancaire.

Vos choix

Vous pouvez exporter vos données, supprimer les données du compte présentes sur cet appareil et demander la suppression de votre compte. La procédure détaillée figure sur la page Suppression des données.
